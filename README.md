# Python Weather CLI App

A simple command-line weather application built with Python using the WeatherAPI service.

## Features
- Takes city and country as input
- Fetches real-time weather data using `requests`
- Parses JSON response
- Displays location, weather, wind, atmosphere, rain, and UV details
- Basic error handling for invalid inputs

## Requirements
- Python 3.x
- requests library

## Run
```bash
pip install requests
python main.py
